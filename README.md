# CASA0006 Road Safety Coursework

This repository contains data and notebook for the CASA0006 coursework project on 
severe road collisions in London.

## Files
- `CASA0006_road_safety_london.ipynb` — main analysis notebook
- `data/dft-road-casualty-statistics-collision-2024.csv` — DfT Road Safety Open Data, 2024
- `data/dft-road-casualty-statistics-road-safety-open-dataset-data-guide-2024.xlsx` — official data guide

## Reproducing
The notebook reads the two data files directly via the GitHub raw URLs, so 
running `Restart & Run All` on a standard Anaconda environment is sufficient 
(scikit-learn, pandas, matplotlib, seaborn, openpyxl).
